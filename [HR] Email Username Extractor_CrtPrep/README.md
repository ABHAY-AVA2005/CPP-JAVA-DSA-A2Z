# Email Username Extractor-CrtPrep

### Difficulty: Medium

## Description

In a system, you are given an email address.

Your task is to extract the username (the part before @).

Input Format

A single string S representing an email address

Constraints

1 ≤ length of S ≤ 10⁵ The string contains exactly one @ Email contains only lowercase letters, digits, and special characters like .

Output Format

Print the username (substring before @)

Sample Input 0

chary123@gmail.com


Sample Output 0

chary123


Sample Input 1

john.doe@yahoo.com


Sample Output 1

john.doe
