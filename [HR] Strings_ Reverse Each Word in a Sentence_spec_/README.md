# Strings: Reverse Each Word in a Sentence(spec)

### Difficulty: Medium

## Description

You are given a sentence consisting of multiple words separated by a single space.

Your task is to reverse each individual word in the sentence while maintaining the original order of the words.

⚠️ Note:

Do NOT reverse the entire sentence
Only reverse characters of each word
Maintain spacing exactly as given


Input Format

A single line containing a string S

Constraints

1 ≤ length of string ≤ 10⁵ String contains alphabets and spaces only Words are separated by exactly one space

Output Format

Print a single line where each word is reversed but the word order remains the same

Sample Input 0

hai iam krishna from CSITECEEEE branch


Sample Output 0

iah mai anhsirk morf EEEECETISC hcnarb


Sample Input 1

hello everyone


Sample Output 1

olleh enoyreve
