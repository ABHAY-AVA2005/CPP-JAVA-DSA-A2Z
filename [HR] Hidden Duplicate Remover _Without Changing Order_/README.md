# Hidden Duplicate Remover (Without Changing Order)

### Difficulty: Medium

## Description

Remove duplicate characters from a string without changing the order of first occurrence.

You are NOT allowed to use collections (Set, Map, etc.)

Input Format

A string S

Constraints

Case-sensitive Only loops & basic logic allowed

Output Format

String with duplicates removed

Sample Input 0

programming


Sample Output 0

progamin


Sample Input 1

banana


Sample Output 1

ban
