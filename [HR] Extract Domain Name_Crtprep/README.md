# Extract Domain Name-Crtprep

### Difficulty: Medium

## Description

You are given an email address.

Your task is to extract the domain name (the part between @ and .)

Input Format

A single string S representing an email address

Constraints

Proper Email

Output Format

Print the domain name

Sample Input 0

chary123@gmail.com


Sample Output 0

gmail


Sample Input 1

sarfaraz@Yahoo.com


Sample Output 1

Yahoo
