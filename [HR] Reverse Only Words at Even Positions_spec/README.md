# Reverse Only Words at Even Positions-spec

### Difficulty: Medium

## Description

You are given a string s consisting of multiple words separated by spaces.

Your task is to reverse only the words at even positions (1-based index) and keep the other words unchanged.

Input Format

A single line string s

Constraints

1 ≤ length of s ≤ 10^5 String contains only alphabets and spaces Words are separated by a single space

Output Format

Print the modified string after reversing words at even positions

Sample Input 0

hai iam krishna from CSITECEEEE branch


Sample Output 0

hai mai krishna morf CSITECEEEE hcnarb


Sample Input 1

hello world java programming


Sample Output 1

hello dlrow java gnimmargorp
