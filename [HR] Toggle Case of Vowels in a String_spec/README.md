# Toggle Case of Vowels in a String-spec

### Difficulty: Medium

## Description

You are given a string S.

Your task is to:

Toggle the case of only vowels in the string Lowercase vowel → Uppercase Uppercase vowel → Lowercase Keep all other characters unchanged

👉 If the string contains no vowels, print:

 ** No vowels present**


Input Format

A single line string S

Constraints

1 ≤ length of S ≤ 10^5 String may contain alphabets, digits, spaces, and symbols

Output Format

Print the modified string after toggling vowels OR print "No vowels present" if no vowels exist

Sample Input 0

Hello Everyone how are Doing


Sample Output 0

HEllO evEryOnE hOw ArE DOIng


Sample Input 1

JAVA PROGRAM


Sample Output 1

JaVa PRoGRaM
