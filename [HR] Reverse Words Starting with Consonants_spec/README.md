# Reverse Words Starting with Consonants-spec

### Difficulty: Medium

## Description

You are given a string s consisting of multiple words separated by spaces.

Your task is to:

Reverse only those words whose starting character is a consonant Keep words starting with vowels unchanged

Input Format

A single line string s

Constraints

1 ≤ length of s ≤ 10^5 String contains only alphabets and spaces Words are separated by a single space

Output Format

Print the modified string after applying the rule

Sample Input 0

Hello Everyone how are Doing


Sample Output 0

olleH Everyone woh are gnioD


Sample Input 1

Apple Banana Orange Mango


Sample Output 1

Apple ananaB Orange ognaM
