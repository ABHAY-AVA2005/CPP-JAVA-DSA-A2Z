# Reverse Username Only

### Difficulty: Medium

## Description

You are given an email address. Reverse only the username part before the at symbol and keep the domain unchanged.

Input Format

A single string S representing the email address

Constraints

1 <= length of S <= 100000 The string contains exactly one at symbol

Output Format

Print the modified email after reversing the username

Sample Input 0

chary123@gmail.com


Sample Output 0

321yrahc@gmail.com


Sample Input 1

a@b.com


Sample Output 1

a@b.com
