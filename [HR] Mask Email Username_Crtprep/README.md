# Mask Email Username-Crtprep

### Difficulty: Medium

## Description

You are given an email address.

Your task is to:
Keep the first and last character of the username
Replace all characters in between with *


Input Format

A single string S representing an email address

Constraints

Proper Email Id

Output Format

Print the masked email

Sample Input 0

chary123@gmail.com


Sample Output 0

c******3@gmail.com


Sample Input 1

sarfaraz@gmail.com


Sample Output 1

s******z@gmail.com
