# Count Blocks of Same Characters-spec

### Difficulty: Medium

## Description

You are given a string s.

Count how many groups (blocks) of consecutive same characters exist.

EX: aaabbcaaa Explanation:(aaa | bb | c | aaa) Output: 4

Input Format

A single string s

Constraints

1≤∣s∣≤10power5 String s contains only valid characters (can be alphabets, digits, or symbols) Input string will not be empty

Output Format

Print number of blocks

Sample Input 0

aaabbcaaa


Sample Output 0

4


Explanation 0

(aaa | bb | c | aaa)

Sample Input 1

abcd


Sample Output 1

4
