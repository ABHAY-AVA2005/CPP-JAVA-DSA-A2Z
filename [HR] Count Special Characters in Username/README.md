# Count Special Characters in Username

### Difficulty: Medium

## Description

You are given an email address. Count how many special characters are present in the username part. Special characters are those that are not letters and not digits.

Input Format

A single string S representing the email address

Constraints

1 <= length of S <= 100000

Output Format

Print the count of special characters in the username

Sample Input 0

john.doe_123@gmail.com


Sample Output 0

2


Sample Input 1

user.name@yahoo.com


Sample Output 1

1
