# Email Domain Frequency Counter-Crtprep

### Difficulty: Medium

## Description

You are given N email addresses. Your task is to count how many times each domain appears.

A domain is the part after @.

Input Format

First line contains integer N (number of emails) Next N lines each contain a valid email address

Constraints

1 ≤ N ≤ 10⁵ Each email contains exactly one @

Output Format

Print each domain along with its frequency Output should be in alphabetical order of domain

Sample Input 0

5
chary@gmail.com
sarf@yahoo.com
abc@gmail.com
test@outlook.com
xyz@yahoo.com


Sample Output 0

gmail.com 2
outlook.com 1
yahoo.com 2
