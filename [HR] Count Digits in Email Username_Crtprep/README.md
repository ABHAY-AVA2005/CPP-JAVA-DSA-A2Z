# Count Digits in Email Username-Crtprep

### Difficulty: Medium

## Description

You are given an email address.

Your task is to count how many digits are present in the username (before @).

Input Format

A single string S representing an email address

Constraints

Proper Email id

Output Format

Print the count of digits in the username

Sample Input 0

chary123@gmail.com


Sample Output 0

3


Sample Input 1

user2024@yahoo.com


Sample Output 1

4
