# First Repeat Breaker-spec

### Difficulty: Medium

## Description

You are given a string s consisting of lowercase alphabets.

Your task is to remove characters after the first repeating character appears (including that repeated occurrence).

Keep characters only until the first repetition happens

Input Format

A single string s

Constraints

1 ≤ |s| ≤ 10⁵

Output Format

Print the resulting string

Sample Input 0

abcdefa


Sample Output 0

abcdef


Sample Input 1

abcabc


Sample Output 1

abc
