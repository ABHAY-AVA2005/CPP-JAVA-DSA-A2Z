# Alternate Case Reversal Trap

### Difficulty: Medium

## Description

Given a string, reverse only those words whose length is odd, while keeping even-length words unchanged. Maintain original word order.

Input Format

A single line string S

Constraints

1 ≤ length of string ≤ 10⁴ Words are separated by a single space

Output Format

Modified string

Sample Input 0

hello world java code


Sample Output 0

olleh dlrow java code


Sample Input 1

java is fun and cool


Sample Output 1

java is nuf dna cool
