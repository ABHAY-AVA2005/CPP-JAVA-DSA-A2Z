# Alternating Prime Check Sequence

### Difficulty: Medium

## Description

Check if numbers alternate between prime and non-prime.

Input Format

The first line contains an integer n (size of the sequence). The second line contains n space-separated integers representing the sequence.

Constraints

1 ≤ n ≤ 10⁵

Output Format

Print "YES" if the numbers alternate between alternate and non-prime. Otherwise, print "NO".

Sample Input 0

5
2 4 3 6 5


Sample Output 0

YES


Sample Input 1

4
4 5 6 7


Sample Output 1

YES
