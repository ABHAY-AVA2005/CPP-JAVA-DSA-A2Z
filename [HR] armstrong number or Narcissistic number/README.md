# armstrong number or Narcissistic number

### Difficulty: Medium

## Description

Write a program that takes an integer N as input and determines whether it is an Armstrong number

Input Format

A single integer N

Constraints

0≤N≤10^9

Output Format

Print "Yes" if N is an Armstrong number. Print "No" otherwise.

Sample Input 0

153


Sample Output 0

Yes


Sample Input 1

123


Sample Output 1

No
