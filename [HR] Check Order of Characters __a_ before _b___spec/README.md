# Check Order of Characters ('a' before 'b')-spec

### Difficulty: Medium

## Description

You are given a string s consisting only of lowercase characters 'a' and 'b'.

Determine whether all occurrences of 'a' appear before any occurrence of 'b'.

Once a 'b' appears, no 'a' should appear after it.

Input Format

A single line containing string s

Constraints

1≤∣s∣≤10power5

Output Format

Print "true" if condition is satisfied Otherwise print "false"

Sample Input 0

aaabb


Sample Output 0

true


Sample Input 1

ababa


Sample Output 1

false
