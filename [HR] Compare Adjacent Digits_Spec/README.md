# Compare Adjacent Digits-Spec

### Difficulty: Medium

## Description

You are given a string num consisting only of digits (0–9).

Compare each pair of adjacent digits and generate a result string:

'>' → if current digit > next digit '<' → if current digit < next digit '=' → if both digits are equal

Input Format

A single string num

Constraints

2≤∣num∣≤10power5

Output Format

Print the resulting comparison string

Sample Input 0

658488


Sample Output 0

><><=


Sample Input 1

12345
